    Class 5

    The first part of the class returns to the concept of spreadsheet design, completing the cash budget question begun last week. The importance of adequate planning is stressed, as is insuring that the resulting spreadsheet application is both as accurate as possible and works all the time.

    The second half of the class focuses on the use of graphs within spreadsheets. Graphs can be extremely useful as a way of presenting complex data so that it can be more easily understood by the user. Obviously different graphs can be used in different situations, so you as the designer must chose carefully to insure that the most appropriate one is used to present the data. You should also remember to carefully label all axes, points etc so that the graph is immediately understandable to anyone unfamiliar with the data.
