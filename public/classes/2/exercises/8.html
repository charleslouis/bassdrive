    Question 2.8

    Type your name into cell A1 of a spreadsheet in the format `Title + (SPACE) + Firstname + (SPACE) + Surname`

    Use appropriate functions to separate this one long entry into cells as follows:

    TitleB3
    FirstnameB4
    SurnameB5
    1. Initially assume that all parts of the name must be entered. Each element of the name can be of variable length but must be present.
    2. Now assume that the first name may or not be omitted. In another set of cells, use appropriate formulae to separate the elements of the string.
