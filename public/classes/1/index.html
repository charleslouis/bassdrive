    Class 1

    The first session is mainly introductory. We will orientate you about the logistics of the course, how the teaching sessions and help sessions will operate, introduce the course website, what you will need to do to successfully complete the course and the teaching / learning philosophy used.

    We will also start working on Microsoft Excel. Today's objectives include understanding the structure of the spreadsheet, starting to use simple calculations, understanding automatic recalculation, copying formulae and handling percentages.

    By the next session, you should have completed all the exercises shown on the left, as well as the readings handed out in class.

    A useful resource.

    If, for whatever reason, you do not have a copy of Microsoft Office, you can use free software easily to work on this course. OpenOffice, from OpenOffice.org, is available in Windows, Macintosh, and Linux versions.
