    Class 3

    The third session continues building your knowledge of the fundamentals of Microsoft Excel, and in particular the correct use of functions. The class starts by exploring Nested If statements, which allow you to make the spreadsheet decide between three, four or even more options.

    The use of =Lookup ( ) will then be explored, as in certain situations this is more efficient than using = If ( ) (Note that =VLookup ( ) and =Hlookup ( ) are covered as part of class 7.

    Lastly Excel's treatment of dates and times is explored, and some of the time / date functions are examined in detail. By the end of the class, you should be familiar with all the basic building blocks of spreadsheets, and prepared for the consolidation exercise that will be the focus of next weeks class.

    By the next session, you should have returned to question 2.8 and attempted it again if you did not successfully complete it, completed all the exercises shown on the left, as well as the readings handed out in class.
